# ES6 Data Manipulation

## 📚 Learning Objectives

At the end of this project, you should be able to explain, without the help of Google:

### 🔹 Array methods
- How to use `map`, `filter`, and `reduce` on arrays
- Understand how these methods transform, filter, or reduce data in arrays

### 🔹 Typed Arrays
- What typed arrays are
- Why they are used for handling binary and numeric data efficiently
- Examples of different typed array types (Int8Array, Float32Array, etc.)

### 🔹 Data Structures
- Understand the `Set` data structure and its uniqueness property
- Understand the `Map` data structure and key-value storage
- Understand `WeakMap` and `WeakSet` and how they differ from Map and Set (weak references and garbage collection)

---

## 🚀 Summary

This project focuses on modern JavaScript ES6 features used for efficient and clean data manipulation, including functional programming methods and advanced data structures.

---

## 🧠 Key Takeaways

- `map` → transform data
- `filter` → select data
- `reduce` → combine data into a single value
- `Set` → store unique values
- `Map` → key-value storage with flexible keys
- `WeakMap / WeakSet` → memory-efficient collections with weak references
- Typed Arrays → optimized arrays for binary data processing
